This repository documents the design, deployment, and security hardening of a simulated corporate IT environment (`LAB.LOCAL`). 

The project mirrors real-world **offshoring operational standards**, focusing on:
* **Infrastructure:** Windows Server 2022 Active Directory deployment.
* **IAM:** Identity lifecycle (JML), RBAC models, and Least Privilege principles.
* **Security:** Group Policy (GPO) hardening, audit logging, and access control.
* **Operations:** Tier 1/2 support workflows, PowerShell automation, and disaster recovery.
